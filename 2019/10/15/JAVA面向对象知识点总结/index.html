JAVA面向对象知识点总结

Posted on 2019-10-15

第二章 类和对象

1.面向对象概念

2.类的创建

File—>new—>class—>类名（类名使用帕斯卡命名法）
语法格式：public class 类名 {}

3.创建对象

语法格式：
类名 对象名=new 类名();
对象名.属性 ；
对象名.方法名() ;

4.构造方法

​ 方法名必须和类名相同，没有返回值类型
​ 作用：创建对象时执行一些初始化操作

5.构造方法重载

在同一个类中，方法名相同，参数列表不同（个数，顺序，类型）的构造方法

6.构造方法的执行

new 类名();—>执行构造方法
super();—>执行了父类的构造方法

7.static静态修饰符

​ 静态修饰符可以修饰属性，方法，代码块
​ 被static修饰的属性和方法又叫类属性和类方法，类属性和类方 法可以通过类名和对象名调用
​ 类名(对象名).属性; 类名(对象名).方法名;

第三章 面向对象三大特征—封装

1.封装的概念

2.封装的步骤

​ 1.尽量把权限降到最低，使用private修饰属性
​ 2.定义setter和getter方法，设置取值范围
​ 3.在main方法中调用setter和getter方法

3.封装的作用

​ 1.提高安全性
​ 2.实现隐藏细节
​ 3.提高复用性

4.修饰符

public—>公共的：在一个项目中都可以访问，权限最高
private—>私有的：只能在本类中进行访问，权限最低
protected—>安全的，受保护的：同一个包能进行访问，继承中之类可以访问父类
default—>默认的：同一个包能访问

第四章 面向对象三大特征—继承

1.继承的概念

父子关系，儿子可以继承父亲的财产；
西虹市首富，王多鱼继承二爷的财产

2.继承的关键字

extends

3.继承的语法

4.使用，在之类中

可以直接调用父类的非私有属性和方法
可以用super.方法 super.属性

5.父类构造方法调用：super();

调用父类的构造方法必须在之类中调用
必须实在之类构造方法的第一行代码

6.重写

必须继承
方法名一致，参数个数，顺序，类型全部一致

第五章 面向对象—多态

1.多态的概念

2.多态的必要条件

​ 1.继承
​ 2.重写
​ 3.父类引用指向子类对象（向上转型）

3.多态的使用

4.向下转型

第六章 抽象

1.抽象的概念

2.抽象的关键字

abstract:可以修饰类，可以修饰方法

3.抽象的用法

//抽象类

//当父类是抽象类的时候，子类必须重写父类里面的抽象方法

4.final修饰符：最终的

修饰变量，修饰方法，修饰类
1.当修饰变量时，变量转换成常量，常量的值不能改变
final int a=18;//a=18;不能改变,必须给一个初始值
2.当修饰方法时，该方法不能被重写
3.当修饰类时，该类不能被继承

第七章 接口

1.接口的概念

2.接口的关键字

3.接口的语法

public interface 接口名{}
File—>new—>interface //创建接口

4.接口的用法

实现接口的关键字 implements

注意接口可以进行多继承
public interface 接口名 extends 接口名，接口名{}
一般用法
public class 类名 extends 父类 implements 接口，接口{}

第八章 集合框架

1.ArrayList

用法
ArrayList list=new ArrayList();
list.add();//添加数据
list.get();//获取数据

2.Map

用法
Map map=new HasMap();
map.put(key,value);//key不能重复
map.get(key);//获取key对象的value

3.泛型

用法
ArrayList list=new ArrayList>();
Map<String,String> map=new HasMap<String,String>();
ArrayList<Map<String,String>> list=new ArrayList> <Map<String,String>>();
